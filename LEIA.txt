1. Para rodar o React Native
Primeiro, instalamos a base do React Navigation. Cole o seguinte comando no terminal e aperte Enter:
npm install @react-navigation/native

2. Instalando as dependências de tela
npx expo install react-native-screens react-native-safe-area-context


3. Como no nosso código de exemplo nós usamos o createNativeStackNavigator (que é o modelo de navegação que "empilha" uma tela sobre a outra,
 permitindo o efeito do botão de "voltar"), precisamos instalar esse pacote específico também:
 npm install @react-navigation/native-stack

 4. Instalação do AsyncStorage
 npm install @react-native-async-storage/async-storage

5. Instalação do blur (efeito de vidro fosco)
npx expo install expo-blur

 Tudo pronto! 🚀 (Por enquanto)
